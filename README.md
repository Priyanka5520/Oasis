# 💻 Java Mini Projects 


\

---

## 📌 Overview

This repository contains two Java-based console applications developed as part of my internship application for **Oasis Infobyte**. These projects demonstrate strong fundamentals in Java, Object-Oriented Programming, and problem-solving.

---

## 📚 Project 1: Digital Library Management System

### 🔹 Features:

* Add, view, and search books
* Issue and return books
* Unique book ID system
* Input validation

### 🖼️ Sample Output:

```
==== Digital Library Management ====
1. Add Book
2. View All Books
3. Search Book
4. Issue Book
5. Return Book
```

---

## 🏧 Project 2: ATM Interface System

### 🔹 Features:

* Secure login using Account Number & PIN
* Balance inquiry
* Deposit & withdraw money
* Error handling for invalid inputs

### 🖼️ Sample Output:

```
🏧 ATM Interface
Enter Account Number:
Enter PIN:
✅ Login successful!
```
